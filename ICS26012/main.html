First Come First Serve

FCFS is a non-preemptive scheduling algorithm that prioritizes processes based on their arrival order, with the first arriving process being the first to be executed.

Try FCFS Algorithm
Round-Robin

This is a preemptive scheduling algorithm where each process is assigned a fixed time quantum, and the CPU switches between processes in a circular order until all processes are completed.

Try Round-Robin Algorithm
Scan

Also known as the Elevator algorithm, this is a disk scheduling algorithm that services requests in a one-way, back-and-forth motion, like a scanning motion of an elevator.

Try Scan Algorithm
